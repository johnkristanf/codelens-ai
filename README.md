# CodeLens AI

CodeLens AI is an intelligent VS Code extension designed to help developers seamlessly understand, explore, and document their projects directly from their editor.

## Key Features

- **Answers Codebase-Related Questions:** Instantly get accurate answers to your questions about code logic, feature implementation, and overall structure.
- **Creates Documentation:** Automatically generate clear, comprehensive, and up-to-date documentation for your projects.
- **Generates Architectures & ERDs:** Easily visualize your system with automatically created architecture diagrams and Entity-Relationship Diagrams (ERDs).
